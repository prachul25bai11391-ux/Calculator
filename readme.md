# Scientific Calculator using Python Tkinter

This project involves the development of a fully functional **Scientific Calculator** using **Python** and its standard GUI library, **Tkinter**. The calculator features a user-friendly interface capable of performing both standard arithmetic and advanced scientific calculations.

## üìù Features
* **Standard Operations:** Addition, Subtraction, Multiplication, Division, Modulus.
* **Scientific Operations:**
    * **Trigonometry:** Sin, Cos, Tan (and their hyperbolic variants).
    * **Logarithms:** log, log10, log2, log1p.
    * **Exponentials:** exp, expm1, e.
    * **Others:** Square Root, Pi, Tau, Degree conversions, Gamma function.
* **Dual Modes:** Toggle between 'Standard' (basic view) and 'Scientific' (expanded view) via the File menu.
